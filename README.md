Give Sphere
Give Sphere is a modern charity and donation platform designed to
make giving simple, transparent, and impactful. It connects donors with meaningful causes, allowing them to support initiatives in education, healthcare, disaster relief, and community development.

Features

Secure Donations: Multiple payment options with safe transactions

Campaign Showcase: Explore active charity campaigns with fundraising goals

User Dashboard: Track your donations and see your impact

Admin Dashboard: Manage campaigns, donations, events, and users efficiently

Events & Volunteering: Join events and volunteer for causes

Team & Impact Section: Highlighting the people and stories behind the charity

Newsletter Subscription: Stay updated with the latest campaigns and events

Responsive Design: Fully functional on desktop and mobile devices

Tech Stack

Frontend: HTML, CSS, JavaScript, Bootstrap

Backend: PHP, MYSQL

Database: MySQL

Other Tools: Google Fonts, APIs for maps/events
